proper command- and cntrl- options. At last sight, you had to define command keys using

a LispWorks bug, but surely that must be fixed by now.
I looked at some instances how I used like "mouse-click" and it seems they expected the ordinary
action of a mouse click to be executed. In the case of "mouse-click", the cursor should be moved
to where the mouse is clicked.
In any case, these commands ertainly should not crash or return a weird object.
