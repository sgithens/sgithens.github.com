boxer-iconsets-sketch
AboutWhen a box is shrunk it has a boxtop. Like the top of a box you would see in real life if you closed a box.
There are a few different options for what boxtops can look like. Additionally you can have a boxtop box
or a named graphics box.

We'll used the named graphics option, as well as some high level boxes with graphics of the same names,
using names from the freedesktop.org icon spec, and toggle their exports to swap icons. We are using the
original Boxer DB icons, and some icons from the Elementary OS project, and nice looking modern Linux
distribution.

https://github.com/elementary/icons
https://specifications.freedesktop.org/icon-naming-spec/icon-naming-spec-latest.html
